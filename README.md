## Fish Recognition Model
This model is trained to identify 10 different fish species commonly found in Galveston 

Specifically:
- Red Drum
- Spotted Seatrout
- Black Drum
- Spotted Seatrout
- Southern Flounder
- Southern Kingfish (Whiting)
- Atlantic Croaker
- Gafftopsail Catfish
- Sheepshead
- Ladyfish
- Cervalle Jack

# Prerequisites

 To use new data to train model:

```
pip install -r requirements.txt
```
